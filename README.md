# Program1 - Ingliz tilini o'rgan

Bu loyiha IELTS tayyorgarlik va ingliz tilini o'rganish uchun web-aplikasiyadir.

## Fayllar

- **app.js** - Asosiy dastur logikasi
- **app-ielts.js** - IELTS qismi
- **data.js** - So'zlar va mashqlar bazasi
- **i18n.js** - Til tarjimaları
- **index.html** - Asosiy HTML
- **styles.css** - CSS stillar

## Boshlash

1. `index.html`-ni brauzerda oching
2. Ingliz tilini o'rganishni boshlang
3. IELTS tayyorgarligini davom eting
